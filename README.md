# Salesforcast
# 📊 Sales Forecast Dashboard – Power BI Project

## 🚀 Project Overview

This project is a **Sales Forecast Dashboard** built using **Power BI**. The dashboard helps analyze historical sales data, identify trends, monitor KPIs, and predict future sales performance.

The goal of this project is to provide business insights through interactive visualizations and forecasting techniques.

---

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel / CSV Dataset
* DAX (Data Analysis Expressions)
* Power Query
* Data Visualization Techniques

---

## 📌 Features

* Interactive sales dashboard
* Monthly and yearly sales analysis
* Profit and revenue tracking
* Region-wise sales performance
* Product category analysis
* Forecasting future sales trends
* KPI cards for quick insights
* Dynamic filters and slicers

---

## 📈 Dashboard Insights

The dashboard provides insights such as:

* Top-performing products
* Best sales regions
* Monthly sales growth
* Revenue and profit trends
* Future sales prediction using forecasting
